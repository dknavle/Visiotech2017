PROJECT EXHIBITION



EVENT DESCRIPTION:

In this event students from different collages will give their demonstration on their projects. According to the winning criteria, project winners will be decided. Need of project instruments like CRO, Power supply, function generator will be provided.



RULES:

  1. Maximum 2 or minimum 1 student can participate in a group.
  2. For every group project report will be compulsory.
  3. Students pursuing education in diploma as well as engineering degree are eligible to participate.
  4. The participant should produce valid ID cards.


REQUIREMENTS:
  1. Project output.
  2. Project report.
  3. Contribution in project will be considered.

TEAM AND FEE STRUCTURE:

  1. Team Size : Maximum 2 participants per team.
  2. Registration Fee : Rs 50/- per participant.
