MICROMANIA


DESCRIPTION:

This event is based on “MICROPROCESSOR”. Participants will able to improve their programming skills as well as software implementation techniques.


ROUNDS:

ROUND 1:QUIZ

  1. Quiz competition will be held.
  2. Quiz will be based on simple questions of “MICROPROCESSOR” (8085/8086).
  3. Top scorer will play next round.
  4. This is elimination round.

    5. ROUND 2:CODING

  5. Bug program will be given to one participant and they will debug it.
  6. Second partner will describe the logic behind it.
  7. According to logic behind the program groups will be selected for next round.

    8. ROUND 3:SOFTWARE IMPLEMENTATION

  8. A circuit will be provided to the participants.
  9. Participants have to simulate it on “PROTEUS SOFTWARE”.
  10. One who will simulate first, that group will be awarded.

RULES:


  1. Maximum 2 participant will be allowed.
  2. The given program will be based on “8086 MICROPROCESSOR”.
  3. Software will be provided by the organizers.
  4. This event comprises 3 levels.

    5. Note: Time limit will be specified at the time of event.


TEAM AND FEE STRUCTURE:

Team Size : Maximum 2 participants per team.
Registration Fee : Rs 50/- per participant.
